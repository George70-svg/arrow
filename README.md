# Arrow Game

link: https://george70-svg.github.io/arrow/

## 2D game on canvas

### The game is written in pure JS/TS. The interface is in React

`yarn dev` - For start 

### Stack:
1) React/TypeScript
2) Vite
3) Eslint
4) Prettier
5) FSD
6) SCSS/CSS module
